## Prerequisites

- Access to your Zyro site admin

- Your unique script snippet or widget key from Studio
  - See instructions [here](/help/channels/chat-widget#finding-code-snippet--key)

## Installation Steps

1. **Open your Web site Builder**

2. **At the bottom right, click "Website settings" then "Integrations"**

3. **Paste your unique JavaScript Snippet**

   - Paste in the snippet in to the "Custom code" section

   ```html
   <script
     id="xapp-js"
     src="https://widget.xapp.ai/xapp-chat-widget.js?key=YOUR_CHAT_KEY"
   ></script>
   ```
